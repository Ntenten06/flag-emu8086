# Assembly Flag Project 🇹🇭

การสร้างธงชาติอิตาลีโดยใช้ภาษา Assembly (8086) บนโหมดกราฟิก Video Mode 13h

## Features
โปรเจกต์นี้เป็นการสาธิตการควบคุมในสถาปัตยกรรม x86 เพื่อวาดรูปธงชาติอิตาลี (เขียว, ขาว, แดง)
- Written in x86 Assembly
- Uses BIOS interrupt 10h
- Runs in graphics mode
